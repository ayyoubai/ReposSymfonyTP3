# Symfony TP3 - Product Page & Cart Form

## Description
Ce projet Symfony présente une **page produit interactive** avec un **formulaire d'ajout au panier**.  
Il permet de pratiquer la création d'**entités**, de **contrôleurs**, de **formulaires Symfony** et de **vues Twig**.

## Fonctionnalités principales
- Affichage d'un produit avec image, nom, prix, description et caractéristiques (marque, couleur, connectivité, autonomie).
- Formulaire d'ajout au panier avec sélection de quantité et de couleur, validation des données et bouton d'ajout.
- Messages flash confirmant l'ajout au panier.
- Interface stylisée avec **Bootstrap 5** pour une présentation claire et responsive.
